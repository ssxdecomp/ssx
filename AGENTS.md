# No.

Code that originates purely from a LLM or other agent will not be accepted, and this is not something I'm willing to waver on. This is for a couple reasons:
- LLM generated code is generally of poor to average quality.
- It is very hard to license AI generated code, since the output code may originate from open source or source available code that has varying (including source avaliable, which may actually disallow derivation) licenses. A project like a matching decompilation is already a legal grey line, and I personally don't want to deal with that alongside that grey line already existing.

You may use a LLM or other form of AI to *assist* with decompilation, however.
